GMMProfile

The GMMProfile function takes the output of the function GMM and outputs information about how each cluster diverges from the global data statistics.

GMMProfile Syntax

Version 1.3

SELECT * FROM GMMProfile (
  ON { table | view | (query) } PARTITION BY 1
) AS alias;

GMMProfile Input

The input table is the GMM Output OutputTable, a model file.

GMMProfile Output

Output Table Schema

Column Data Type Description
cluster_id INTEGER Cluster identification number.
dimension VARCHAR Input dimension name.
delta_mean DOUBLE PRECISION Difference between mean of cluster and mean of data set along each dimension.
divergence DOUBLE PRECISION Estimated Kullback-Leibler divergence between cluster and data set along each dimension.
