ShapleyGenerateCombination

The ShapleyGenerateCombination function takes combinations of players and creates input for ShapleyAddOnePlayer.

ShapleyGenerateCombination Syntax

Version 1.3

SELECT * FROM ShapleyGenerateCombination (
  ON { table | view | (query) }
) AS alias;

ShapleyGenerateCombination Input

Input Table Schema

The table can have additional columns, but the function ignores them.

Column Data Type Description
index VARCHAR First column of table. Contains bit strings that indicate which players are in the combination. Length of each bit string is total number of players. Each bit represents a player. Bit has value 1 if player is in the combination and 0 otherwise. Most and least significant bits represent first and last players, respectively. Each row represents a unique combination.
payoff DOUBLE PRECISION Second column of table. Contains observed payoff value for corresponding combination in index column.

ShapleyGenerateCombination Output

Output Table Schema

Column Data Type Description
comb VARCHAR Combination, represented by space-separated player numbers. For example, '1 2' or '2 3 4 5'. First player is number 1, second is number 2, and so on.
Note
For correct Shapley values, the comb column values must be unique.
size INTEGER Combination size.
value DOUBLE PRECISION Assigned value of combination.
